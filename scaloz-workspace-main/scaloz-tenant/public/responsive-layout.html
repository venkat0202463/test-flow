Responsive App Header (Text Wraps Naturally)

Block 1 (Main Content) - Text Wrap Test

This extremely long paragraph is designed to **explicitly test text wrapping** and responsiveness across all device sizes. When the screen width shrinks, this text should fluidly wrap to the next line to prevent horizontal scrolling.

Full width on Mobile (`col-12`), **Half width on Tablet (`col-md-6`)**, One-third on Desktop (`col-lg-4`).

Block 2 (Secondary Content)

This block works with Block 1 to form a two-column layout on medium screens (tablets) before joining the three-column desktop arrangement.

Full width on Mobile (`col-12`), **Half width on Tablet (`col-md-6`)**, One-third on Desktop (`col-lg-4`).

Block 3 (Sidebar/Ad)

On tablets, this block is forced to the next row (since 6+6=12) but integrates perfectly into the three-column desktop layout.

Full width on Mobile/Tablet (`col-12`), One-third on Desktop (`col-lg-4`).

Mobile/Small Screen View: All blocks are stacked full-width.

Tablet/Medium Screen View: Blocks 1 & 2 are in a **2-column layout**, Block 3 is below.

Laptop/Desktop View: All blocks are side-by-side in a **3-column layout**.
